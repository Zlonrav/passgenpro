Руководство пользователя

Что такое детерминизм?
Это значит, что алгоритм работает без случайных чисел. Один и тот же входной набор данных всегда выдает один и тот же пароль. Вам не нужно его сохранять — достаточно запомнить свои ключи.

Как пользоваться?

  • Secret Word 1: Ваша постоянная метка (например, логин или название сервиса).
  • Secret Word 2: Ваше главное секретное слово (мастер-пароль).
  • Контрольное число: Дополнительный множитель для создания разных вариаций пароля (от 0 до 999).

Безопасность

Приложение работает полностью автономно. Ваши данные не передаются по сети и не хранятся в localStorage. Математическая функция FNV-1a преобразует текст в уникальный хэш, который невозможно восстановить обратно.

💡 Совет: Используйте разные контрольные числа для разных аккаунтов, даже если секретные слова одинаковые. Например: 0 для почты, 1 для соцсетей.

Версия алгоритма: 1.0 (Jmnd_V10)
